Introductie

Om mijn Front-End Block te halen heb ik besloten deze website te bouwen. Hierin laat ik zien welke vaardigheden ik inmiddels heb en wat ik kan maken met HTML, CSS en JavaScript. Ik heb meerdere dingen gemaakt die hier in de praktijk te zien zijn en waarvan je zelf de code kunt bekijken aangezien je de files gedownload hebt. Hiermee laat ik zien dat ik voldoende skills heb om het block te halen.

Opdrachten

De punten waaraan ik moet voldoen om het Front-End Block te halen zijn het volgende

HTML

Je hebt onderzoek gedaan naar het werken met HTML5 middels een online cursus. Je kent de basis HTML elementen zoals div, span en p . Je weet wat semantische elementen zijn, en je kan hiermee een basis HTML document opzetten.

CSS

Je kan werken met css selectors: tags, idâ€™s en classes. Je kan werken met css eenheden: %, em, vw, vh en px. Je begrijpt de werking van css display: block, inline-block, inline, flex. Je website is responsive door gebruik te maken van relatieve eenheden, flex, grid, en/of media queries. Je gebruikt geen css framework.

JavaScript

Middels code voeg je nuttige functionaliteit toe aan je applicatie. Je app toont een status en kan reageren op acties van de gebruiker. Je hebt je code zelf geschreven. Je code is leesbaar opgebouwd met heldere functienamen en variabelen, en dit kan je mondeling toelichten. Je gebruikt geen javascript framework of library.

Overig nieuw bewijsmateriaal

Voorderest kun je zelf de code bekijken van deze website. Hier is te zien dat ik genoeg vaardigheden heb om het Block te kunnen halen.
Ik heb de website opgezet met HTML en CSS en heb functionaliteit via JavaScript toegevoegd. Alle code is volledig zelf geschreven, er zijn geen frameworks gebruikt. Alles wat er staat heeft nut, er zijn geen overbodige tags of elementen.
De code is overzichtelijk en in het Engels (ookal zijn de teksten zelf natuurlijk in het Nederlands).
Om de website responsive te maken heb ik gebruik gemaakt van media queries. (Je mag dit uittesten met de developer tool van Chrome).
